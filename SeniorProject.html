Senior Project - Midterm Progress

Digital Products + Experiences

Project Plan:My plan for this project is to provide college students valuable information about the best restaurants at select places I have traveled to throughout the world via a website. I am doing this by sketching out the layout of my page design then using Adobe Xd I am creating a mockup of what the website is going to look like. When the designing process is completed I will go in and code my website in HTML, CSS and JS.

Targeted Users: College students planning a trip

What makes my design different: My design is different because it incorporates my skills as a designer as well as a traveler. Most people who create the content for travel blogs hire someone to make the website to display the adventures the blogger wants to highlight. I am using my knowledge of HTML, CSS and JS as well as my own photos to create a colorful yet sophisticated website to guide my users into a better travel experience.

What kind of features will your design have and why do you add those features?: My design will feature carousels, animated text, various interactive hover states on text and images, as well as animated images.
